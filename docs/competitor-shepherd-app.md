# 경쟁 앱 분석: Shepherd: Spiritual Bible Pet

조사일: 2026-07-19

## 개요

- 개발사: Second Round Inc (창업자 2인)
- 포지셔닝: "The Duolingo for Bible Devotionals" — 게임화된 데일리 묵상 습관 앱
- 콘셉트: 시편 23편 모티프. 사용자가 목자가 되어 자기 영혼을 상징하는 양(램) 캐릭터를 키움
- 실적: Google Play 10만+ 다운로드, 4.7★(18.6K 리뷰) / App Store 4.8★(8.6K 평가), Reference 카테고리 한때 #50
- 수익화: 구독(Super Shepherd, $4.99~59.99) + 무료 체험. 수익의 10%는 선교 후원 표방
- 카테고리: Books & Reference로 등록 (Games 아님 — 경쟁 덜한 차트 노림)

## 핵심 구조: 데일리 3승(Three Daily Wins)

하루에 세 가지를 완료해야 함:
1. 가이드된 성경 구절 읽기
2. 집중 프롬프트로 기도
3. 60초 묵상(리플렉션)

셋 다 완료 → 양이 살아나고 XP 획득, 레벨업. 하루 건너뛰면 양이 기절(faint).

## 기능 상세 (Google Play 설명 기준)

### 현재 제공 기능
| 기능 | 설명 |
|---|---|
| 안내형 성경 읽기 | 하루의 말씀을 따라가며 읽는 구조 |
| 집중 기도 프롬프트 | 사용자 목표에 맞는 기도 템플릿 |
| 60초 묵상/성찰 | 짧게 생각을 정리하는 반성 시간 |
| 반려 양 아바타 성장 | 3과제 완료 시 양이 되살아나고 XP 획득 → 레벨업 |
| 연속 기록 관리 | 연속 수행일, XP, 보석(gems) 등 진행 상태 추적 |
| 듀오링고식 성경 경로 | 학습 경로·진행 상황을 시각화 |
| 원터치 묵상 일기 | 그날 읽은 내용과 연결된 기록 |
| 오프라인 사용 | 인터넷 없이도 읽기·기도 가능 |

### 진행 보상 구조
하루 3과제(성경 구절 읽기 / 집중 기도 / 60초 묵상)를 모두 끝내면 양이 회복하고 XP를 얻으며 성장. 하루 건너뛰면 양이 기절 → 꾸준함을 강하게 유도.

## Pixel KJV에 적용 가능한 배울점

### 1. 데일리 루프를 펫에 결속
스트릭 숫자보다 "안 하면 캐릭터가 쓰러진다"는 감정적 책임감이 리텐션 동력. 리뷰에서 사용자들이 Finch(웰니스 펫 앱)와 비교하며 긍정 평가. 픽셀 아트 방향성과 펫 캐릭터는 궁합이 좋음 — 다마고치 계보.
→ [pixel-design 방향](feedback_pixel_design.md 참고, 프로젝트 메모리)과 결합해 픽셀 동반자 캐릭터 + 데일리 루프(오늘의 구절 암송 + 복습 N개) 도입 검토 가치 있음.

### 2. 포지셔닝 한 줄 문구
스토어 첫 스크린샷 캡션이 "The Duolingo for Bible Devotionals" 한 줄. 스토어·랜딩·SNS에서 동일 문구 반복.
→ Pixel KJV도 "성경 암송의 듀오링고" 류의 한 줄 포지셔닝을 스토어/랜딩에 일관되게 노출 검토.

### 3. ASO 스크린샷 디테일
기능 나열이 아니라 혜택 문구("Prayers that fill your cup", "Customize any Bible verse")를 큰 글씨로. 장표마다 배경색 교체해 캐러셀에서 시선 끔. iOS 부제는 키워드 포함("Daily Devotional Fellowship").

### 4. "Coming Soon" 예고 전략
AI Bible Chat, 친구 스트릭 공유, 희귀 스킨을 리스팅에 미리 예고해 기대감·재방문 이유 조성. 출시 전 디스코드 커뮤니티 운영 + 크리에이터 제휴 페이지(/creators) 운영.

### 5. 신앙 앱 특유의 신뢰 방어막
"For Christians, By Christians", "수익 10% 선교 후원" 문구로 "신앙으로 돈 버냐" 반감 완화. (실제로 최상단 불만 중 하나가 "영성은 무료여야" 류)

## 반면교사 — 하지 말아야 할 것

- **과한 페이월**: 무료 버전 제한이 심하고 구독 화면이 자꾸 떠서 앱을 못 쓴다는 불만 반복. Duolingo와 비교해 "접근성이 좋아야 스트릭이 유지된다"는 지적이 정곡.
  → 코어 루프(암송·복습)는 무료로 열고, 스킨/꾸미기 등 부가 요소에 과금하는 편이 안전.
- **버그가 루틴을 깨면 치명적**: 가입 실패, 스크롤 안 됨, 프리즈 등이 "아침 경건 시간 방해"로 직결되는 혹평. 매일 반복하는 앱은 코어 루프 안정성이 신규 기능보다 우선.
- **번역본 요청 미대응**: KJV 등 특정 번역본 요청이 많은데 못 채움 → KJV 전문 앱인 Pixel KJV의 차별점 포인트.

## 우선순위 제안

가장 적용 가치 높은 조합: **펫(동반자 캐릭터) + 데일리 3승 루프**. 기존 코스/목숨 시스템 위에 얹기 자연스럽고 픽셀 아트 방향성과 시너지 큼.

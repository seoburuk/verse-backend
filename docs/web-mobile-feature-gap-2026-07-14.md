## 결론

- **모바일이 웹에 뒤처진 항목:** 영어 i18n, 엔터 제출, 암송 중 목숨 표시, 뒤로가기 목숨 차감, **type 모드 틀림 시각효과**, 유저네임 수정, 이모지→픽셀아트, 대시보드 정렬·그룹핑·0숨김, 암송화면 진행바, 모드 선택 영속화.
- **웹이 모바일에 뒤처진 항목:** 목숨 소진 보상광고.
- **양쪽 다 없는 항목(신규):** 비밀번호 찾기, 이메일 인증, 30일 연속 성취, 2줄 이상 자동 드래그, 랭킹 전면광고, 많이 외운 순 정렬, 딕테이션 밑줄, 암송 뒤로가기 시 모드선택 복귀.
- **문서에서 특별히 지적한 문제** — "모바일에서 틀렸을 때 키캡 이펙트가 없어 구별이 안 됨": 확인됨. drag 모드만 쉐이크가 있고 type/dictation 모드엔 색상 변화만 있어 웹 대비 피드백이 약함.

우선순위:
1. type/dictation 모드 틀림 시각효과 (최우선)

3. 암송 중 상단 목숨 표시
4. 뒤로가면 목숨 차감
5. 대시보드 정렬·0숨김·권별 그룹핑

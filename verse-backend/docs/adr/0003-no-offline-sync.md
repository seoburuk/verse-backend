# ADR 0003: 오프라인 양방향 동기화 안 함

## 상태
채택

## 맥락
암송은 이동 중·새벽기도 등 네트워크 불안정 상황에서 많이 일어난다.
완전 오프라인을 지원하면 양방향 sync + 충돌 해결이 필요 → 복잡도 폭증.

## 결정
양방향 sync 미지원. 대신 코스 본문만 "읽기 캐시"로 미리 내려받음.
진도 쓰기는 온라인일 때만 전송.

## 근거
- 읽기 캐시는 충돌이 없다(읽기 전용).
- 개인 프로젝트 범위 관리: sync 엔진은 미완성의 지름길(기획서 §0, §11).

## 트레이드오프
- 오프라인 중 한 시도 결과는 즉시 서버 반영 안 됨. 재연결 시 전송 정책 필요.
